Overview

MCORDER is a comprehensive web-based ordering system designed to streamline the procurement process of non-raw materials for McDonald's Manager Stock and Vendors. This project was developed as part of the Capstone Design course using Agile Scrum methodology.

The system addresses the challenges in managing orders between McDonald's outlets and their suppliers, providing a structured platform for material requests, approvals, and tracking with comprehensive documentation throughout the ordering lifecycle.

Project Goals

Streamlined Procurement

Simplify the ordering process for non-raw materials between outlets and vendors

Better Documentation

Maintain comprehensive records of all orders and transactions for accountability

Multi-Role System

Support different user roles for managers, stock handlers, and vendors

Key Features

01
