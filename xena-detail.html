Overview

XENA is an initial redesign project for the work order management system, originating from my user research during my internship. Starting with comprehensive interviews and direct field observations, I identified key pain points experienced by agents to create a more user-friendly and sustainable application.

I am currently continuing this project as my final thesis (Skripsi), implementing the Prototyping methodology for web development. Several new features have been added beyond the initial internship scope to specifically adapt to the XENA Workorder application's workflow and operational needs.

Project Goals

Enhanced User Experience

Redesign interface to be more intuitive and user-friendly for field technicians and managers

Operational Efficiency

Streamline work order processes to reduce processing time and improve productivity

Real-time Tracking

Implement real-time status updates and monitoring for better visibility
